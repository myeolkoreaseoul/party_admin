## 🎨 HTML → Figma 가져오기 가이드

### ✅ 단계별 방법

#### Step 1: 이 페이지 배포하기
1. 위쪽 **"Publish" 탭** 클릭
2. 원클릭 배포
3. 생성된 URL 복사 (예: https://your-site.com)

#### Step 2: Figma 플러그인 설치
1. Figma 열기
2. **Plugins → Browse plugins in Community** 메뉴
3. 다음 중 하나 검색 및 설치:
   - **"html.to.design"** ⭐ (가장 추천)
   - **"Figma Import"**
   - **"URL to Figma"**

#### Step 3: HTML을 Figma로 Import
1. Figma에서 새 파일 생성 (또는 기존 파일 열기)
2. 플러그인 실행: **Plugins → html.to.design**
3. 배포된 URL 입력
4. **Import** 클릭
5. 자동으로 Figma 레이어로 변환됩니다! ✨

#### Step 4: Figma에서 자유롭게 수정
- 이미지 교체 (드래그 앤 드롭)
- 색상 변경
- 텍스트 수정
- 요소 추가/삭제
- 레이아웃 조정

#### Step 5: 수정 완료 후
- Figma에서 스크린샷 또는 Export
- 저한테 공유
- 제가 HTML 업데이트

---

### 대안: 지금 바로 Figma에 붙여넣기

**만약 배포 전에 바로 시도하고 싶으시면:**

1. Figma 플러그인 **"html.to.design"** 실행
2. **"Paste HTML"** 옵션 선택
3. 아래 HTML 코드 전체 복사해서 붙여넣기
4. Figma로 변환 완료!

---

### 🎯 가장 빠른 방법 요약

```
현재 상태
↓
Publish 탭에서 배포 (1분)
↓
Figma → html.to.design 플러그인
↓
URL 입력 → Import
↓
Figma에서 수정
↓
스크린샷 공유
↓
HTML 업데이트 완료!
```
