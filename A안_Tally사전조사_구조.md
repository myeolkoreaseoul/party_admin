# A안: Tally 사전조사 양식 구조

## 목표
초대장 신청자가 프립 예약 후, **사전조사**에서 **자격 인증 자료**를 제출하도록 유도하여,  
허위 신청을 방지하고 고급 인력만 최종 승인하는 시스템 구축.

---

## 📋 양식 구조

### 1. 기본 정보
1. **이름** (필수, Text Input)
2. **연락처** (필수, Phone or Email)
3. **성별** (필수, Multiple Choice: 남성/여성/기타)
4. **나이** (필수, Number)

---

### 2. 🔥 인증 섹션 (핵심!)

#### Q1. 초대장 신청 시 선택하신 자격은 무엇인가요?
- **타입**: Multiple Choice (복수 선택 가능)
- **옵션**:
  - ☐ 직업 (전문직/대기업/공기업/공무원)
  - ☐ 학력 (SKY, 해외대학 QS 200위 이내)
  - ☐ 연봉/자산 (연봉 8천만원 이상, 연매출 2억 이상, 자산 5억 이상)
  - ☐ SNS 인플루언서 (팔로워 5천명 이상)
  - ☐ 운동 (하이록스/크로스핏/마라톤 완주 또는 체지방 15%/23% 미만)
  - ☐ 외모 (승무원/아나운서/모델/미인대회 등)
  - ☐ 기존 파티 득표 3회 이상
  - ☐ 기타 내부 추천인

---

#### Q2. 인증 자료 제출 (조건부 필수)

##### 1) **직업** 선택 시
- **파일 업로드**: "명함 또는 재직증명서"
- **조건부 표시**: Q1에서 "직업" 선택 시만 표시
- **필수 여부**: 선택 (권장)

##### 2) **학력** 선택 시
- **파일 업로드**: "졸업증명서 또는 학생증"
- **조건부 표시**: Q1에서 "학력" 선택 시만 표시

##### 3) **연봉/자산** 선택 시
- **파일 업로드**: "소득증명원 또는 통장 잔고 (선택)"
- **조건부 표시**: Q1에서 "연봉/자산" 선택 시만 표시
- **필수 여부**: 선택

##### 4) **SNS** 선택 시
- **텍스트 입력**: "인스타그램 아이디 (@ 포함)"
- **조건부 표시**: Q1에서 "SNS" 선택 시만 표시
- **필수 여부**: 필수
- **예시**: `@your_instagram_id`

##### 5) **운동** 선택 시
- **파일 업로드**: "완주 인증서 (선택)"
- **조건부 표시**: Q1에서 "운동" 선택 시만 표시

##### 6) **외모** 선택 시
- **파일 업로드**: "전신 사진 3장 (정면, 측면, 자유 포즈)"
- **조건부 표시**: Q1에서 "외모" 선택 시만 표시
- **필수 여부**: 필수
- **최대 파일 수**: 3개

##### 7) **득표** 선택 시
- **텍스트 입력**: "이전에 참석한 파티명 (3개)"
- **조건부 표시**: Q1에서 "득표" 선택 시만 표시

##### 8) **추천인** 선택 시
- **텍스트 입력**: "추천인 이름 및 연락처"
- **조건부 표시**: Q1에서 "추천인" 선택 시만 표시
- **필수 여부**: 필수

## 🔧 Tally 설정 방법

### Step 1: Tally 폼 편집 화면 열기
1. https://tally.so/dashboard 접속
2. 기존 사전조사 폼 선택 또는 새로 생성

---

### Step 2: 인증 섹션 추가
1. **섹션 추가**: "🔥 자격 인증 (필수)"
2. **Multiple Choice 추가**:
   - 질문: "초대장 신청 시 선택하신 자격은?"
   - 옵션: 위 8가지 (직업, 학력, 연봉, SNS, 운동, 외모, 득표, 추천인)
   - **복수 선택 허용** (중요!)

---

### Step 3: 조건부 필드 추가
#### 직업 선택 시
1. **File Upload 필드** 추가
2. 레이블: "명함 또는 재직증명서 업로드"
3. **조건부 표시 설정**:
   - Show this field **when** "초대장 신청 시 선택하신 자격은?" **contains** "직업"
4. 파일 타입: Images, PDF

#### SNS 선택 시
1. **Text Input 필드** 추가
2. 레이블: "인스타그램 아이디 (@ 포함)"
3. **조건부 표시 설정**:
   - Show this field **when** "초대장 신청 시 선택하신 자격은?" **contains** "SNS"
4. **필수 설정** (Required)

#### 외모 선택 시
1. **File Upload 필드** 추가
2. 레이블: "전신 사진 3장 (정면, 측면, 자유 포즈)"
3. **조건부 표시 설정**:
   - Show this field **when** "초대장 신청 시 선택하신 자격은?" **contains** "외모"
4. 파일 타입: Images
5. **최대 파일 수**: 3
6. **필수 설정** (Required)

---

### Step 4: 저장 및 공유 링크 생성
1. **Save** 클릭
2. **Share** → **Get Link**
3. 링크를 복사하여 **사전조사 발송 시 사용**

---

## 📨 사전조사 발송 시점

### 언제 발송?
- **프립 예약 완료 후** (Tasker로 프립 카톡 감지 시)

### 발송 방법
- **이메일**: "프립 예약이 확인되었습니다. 사전조사를 작성해주세요: [링크]"
- **문자**: "프립 예약 완료! 사전조사 작성 필수: [링크]"

---

## 🎯 최종 승인 기준

### 자동 최종 승인 조건
- 사전조사 완료 ✅
- 인증 자료 제출 ✅
- 인증 자료 검증 통과 ✅

### 환급 불가 조건
- 인증 자료 미제출
- 인증 자료 미흡 (흐릿한 사진, 불명확한 서류 등)
- 허위 신청 (다른 사람 자료 제출 등)

## 💡 참고: Google Sheets 연동

Tally는 Google Sheets와 자동 연동됩니다:
1. Tally → Integrations → Google Sheets
2. 사전조사 응답이 자동으로 "사전조사" 시트에 기록됨
3. Apps Script의 `통합고객관리업데이트()` 함수가 자동으로 병합
