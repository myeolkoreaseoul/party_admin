# Figma로 가져갈 때 필요한 것들

## 방법 1: 배포된 URL 사용 (추천)
- Publish 탭에서 배포
- URL을 Figma html.to.design 플러그인에 입력

## 방법 2: HTML 파일 다운로드
- index.html 다운로드
- Figma 플러그인에서 "Upload HTML file" 옵션 사용

## 방법 3: HTML 코드 복사
- index.html 내용 전체 복사
- Figma 플러그인에서 "Paste HTML" 옵션 사용

---

## 🎨 Figma 플러그인 추천 순서

1. **html.to.design** ⭐⭐⭐⭐⭐
   - 가장 정확한 변환
   - URL, HTML 코드, 파일 모두 지원
   
2. **Figma Import**
   - 간단한 UI
   - URL 입력만 지원

3. **Anima - Export to Code**
   - 역방향도 가능 (Figma → HTML)

---

## ✅ 작업 흐름

```
현재 HTML
  ↓
Publish (배포)
  ↓
Figma html.to.design 플러그인
  ↓
Figma 레이어로 변환
  ↓
이미지/색상/텍스트 수정
  ↓
스크린샷 또는 Export
  ↓
저한테 공유
  ↓
HTML 업데이트
  ↓
완성! 🎉
```
