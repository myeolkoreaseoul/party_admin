🎨 랜딩페이지 디자인 수정 가이드

✅ 추천 방법: Figma에서 수정 → HTML 재변환

Step 1: 원본 Figma 파일 수정
- 기존 Figma 디자인 열기
- 이미지, 색상, 텍스트 모두 수정
- 레이아웃도 자유롭게 변경
Step 2-A: 스크린샷으로 공유 (가장 쉬움)
- 전체 페이지 스크린샷 (Cmd/Ctrl + Shift + 4)
- 또는 섹션별 고해상도 스크린샷
- 저한테 업로드
- 제가 픽셀 퍼펙트하게 HTML/CSS로 재구현
Step 2-B: Figma 플러그인 사용
- Locofy.ai 또는 Anima 플러그인
- HTML/CSS 코드 Export
- 저한테 공유
- 제가 최적화 및 기능 연동

🔄 대안: HTML을 Figma로 가져오기

Step 1: Figma 플러그인 설치
- Figma 열기
- Plugins → Browse plugins
- html.to.design 검색 및 설치
Step 2: HTML 가져오기
- 플러그인 실행
- 옵션 1: 배포된 URL 입력
- 옵션 2: HTML 코드 직접 붙여넣기
- 자동으로 Figma로 변환됨!
Step 3: Figma에서 수정
- 이미지 교체
- 색상 변경
- 레이아웃 조정
- 텍스트 수정
Step 4: 수정본 공유
- 스크린샷 또는 Figma 링크
- 저한테 공유
- 제가 HTML로 재작업

⚡ 가장 빠른 방법: 구체적으로 요청

이렇게 말씀해 주시면 바로 수정해드려요:

"헤더 배경색을 #000000으로 바꿔줘"
"골드 색상을 #E8B923으로 바꿔줘"
"프로그램 1부 시간을 13:00~15:00으로 바꿔줘"
"HOST 은비 설명을 XXX로 바꿔줘"
"이 이미지를 [URL]로 교체해줘"
"프로그램 섹션을 호스트 섹션보다 아래로 옮겨줘"
"폰트를 Pretendard로 바꿔줘"

→ 즉시 수정 완료!

💡 제 추천

1단계: 원본 Figma 디자인을 완벽하게 수정하세요
- 이미지 모두 교체
- 색상 조정
- 텍스트 수정
- 레이아웃 완성

2단계: 완성된 Figma 디자인을 저한테 공유
- 전체 스크린샷 (가장 추천)
- 또는 섹션별 스크린샷
- 또는 Dev Mode CSS 정보

3단계: 제가 완벽하게 HTML/CSS로 재구현
- 픽셀 퍼펙트 구현
- Google Apps Script 연동
- 배포 준비 완료

이 방법이 가장 정확하고 빠릅니다! 🚀
